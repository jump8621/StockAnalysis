STOCK PRICE AND PERFORMANCE ANALYSIS

An Introduction to Analyzing a Comparative Stock Portfolio using Data Visualizations

Project Goals:

This is the first part of our final group Project for the UNC-Chapel Hill Data Analytics Bootcamp.

  • Project 2a: We gathered stock price data for five aeropspace stocks; all of which are large-cap U.S. equities. We sourced such data for a chosen (3 year) period ending 12/31/20.
  • Project 2b: We then used various data coding methodologies to measure the relative performance of the chosen portfolio; against each other and an apppropriate large-cap equity market index.
  • Project 3: We then plan to determine which machine learning techniques are best at predicting financial data and stock performance.

    Specific Project Objectives:

  • Utilize search engines via APIs and use various information retrieval techniques for searching historical stock data.
  • Use a web application to produce numeric summary tables of historical stock price, peformance and stock volume for the 3 year test period for the chosen portfolio.
  • Based on the preceding, provide a visualization of the historical and comparative performance of the 5 similar aerospace stocks.

    Process Steps:

  • Utilize search engines (https://import statsmodels.api and https://yahoo-finance-api) to use information retrieval techniques for searching historical stock data.
  • We then queried our database for the stock symbols for our chosen stocks and used the APIs to download the historical price and volume data for each symbol from 1/1/2010 to 12/31/20.
  • The stocks were also queried for a more recent three year period (2018-2020).
  • We converted data to a pandas dataframe and coded the information in Junyper Notebook for further analysis.
